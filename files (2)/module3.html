Module 3: Cascading Style Sheets

This module focuses on CSS, which is used to style and lay out web pages built with HTML.

Topics Covered

  • Selectors: element, class, and ID selectors
  • The box model: margin, border, padding, content
  • Layout techniques: Flexbox and Grid
  • Responsive design with media queries
  • Using CSS variables for consistent theming

Example Snippet

This site's cards use a simple class selector:

.module-card {
  background-color: #ffffff;
  border-radius: 10px;
  padding: 1.5rem;
}

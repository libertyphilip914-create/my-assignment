Module 1: Web Programming Basics

This module introduces the theoretical foundations of the internet and the web, and explains why web programming matters in today's digital world.

Topics Covered

  • How the internet works: clients, servers, and protocols
  • The difference between the internet and the web
  • What a web browser does
  • Front-end vs. back-end programming
  • Why web programming skills matter

Key Takeaway

Before writing a single line of HTML or CSS, it helps to understand the bigger picture: how a request travels from your browser to a server and back, and where your code fits into that journey.

Module 2: Hypertext Markup Language

This module covers the basics of HTML, the language used to structure content on the web.

Topics Covered

  • Document structure: <html>, <head>, <body>
  • Common tags: headings, paragraphs, lists, images
  • Attributes and how they modify elements
  • Links and navigation with <a>
  • Building forms with <form>, <input>, and <label>

Example Snippet

A basic HTML link looks like this:

<a href="module3.html">Go to Module 3</a>
